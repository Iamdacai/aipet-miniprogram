## 🚀 部署指南

### 1. 配置 AppID
修改 `project.config.json`:
```json
{
  "appid": "你的小程序 AppID"
}
```

### 2. 配置域名
微信公众平台 → 开发管理 → 开发设置 → 服务器域名
- request 合法域名：`https://caiyuyang.cn`

### 3. 编译运行
1. 微信开发者工具导入项目
2. 选择 AppID（或测试号）
3. 编译运行

### 4. 上传发布
1. 点击「上传」按钮
2. 填写版本号和备注
3. 提交审核
4. 审核通过后发布
