# TabBar 图标

请在此目录放置以下图标文件（尺寸：81x81px）：

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `pet.png` - 宠物图标（未选中）
- `pet-active.png` - 宠物图标（选中）
- `reminder.png` - 提醒图标（未选中）
- `reminder-active.png` - 提醒图标（选中）
- `article.png` - 文章图标（未选中）
- `article-active.png` - 文章图标（选中）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中）

**图标要求：**
- 格式：PNG
- 尺寸：81x81 像素
- 背景：透明
- 颜色：
  - 未选中：#999999
  - 选中：#4A90D9

可以使用 [iconfont](https://www.iconfont.cn/) 或 [Flaticon](https://www.flaticon.com/) 下载图标。
